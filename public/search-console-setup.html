Remove Chrome “Dangerous site” warning

This page explains how to request a Google Safe Browsing review for this portfolio demo. Code changes alone cannot remove the red Chrome interstitial — Google must approve the site.

Your URLs

  • Production: master-app-ten.vercel.app
  • Clean demo alias: apexbank-portfolio-demo.vercel.app (same deployment; use while waiting for review)
  • About this demo: /about-demo.html

Step 1 — Verify in Google Search Console

Choose either method below.

Option A — HTML file (recommended)

  1. Open Google Search Console
  2. Add property: https://master-app-ten.vercel.app
  3. Choose HTML file verification
  4. Google shows a filename like google1234567890abcdef.html — copy the name without .html
  5. In this project folder, run:
    npm run gsc:verify -- google1234567890abcdef
  6. Deploy: npx vercel deploy --prod
  7. Click Verify in Search Console

Option B — HTML meta tag

  1. In Search Console, choose HTML tag verification
  2. Copy the content="..." value from the meta tag Google shows
  3. Run:
    npm run gsc:verify -- --meta YOUR_TOKEN_HERE
  4. Deploy: npx vercel deploy --prod
  5. Click Verify in Search Console

Step 2 — Request security review

  1. In Search Console → Security & manual actions → Security issues
  2. Confirm fixes are live (demo banners, about page, noindex, non-deceptive copy)
  3. Click Request review
  4. Paste the text below into the explanation field
